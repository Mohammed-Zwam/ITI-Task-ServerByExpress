Product API Endpoints

GET /api/products

Retrieve all products

Example Request:

GET http://localhost:3000/api/products

GET /api/products/:id

Retrieve a single product by ID

Example Request:

GET http://localhost:3000/api/products/8

POST /api/products

Create a new product

Example Request:

POST http://localhost:3000/api/products/add

Request Body:

{
    "name": "New Product",
    "price": 99.99,
    "description": "Product description"
}

PUT /api/products/:id

Update an existing product

Example Request:

PUT http://localhost:3000/api/products/update/8

Request Body:

{
    "name": "Updated Product",
    "price": 109.99
}

DELETE /api/products/:id

Delete a product

Example Request:

DELETE http://localhost:3000/api/products/delete/8
